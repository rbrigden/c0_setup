# c0-setup
An unofficial setup utility for C0, a subset of C used to teach 15-122 @ CMU. Get the gem at [c0-setup](https://rubygems.org/gems/c0-setup).

## Installation

For Mac or Linux:

```bash
$ gem install c0_setup
```
 
## Setup cc0 and coin  ##

```bash
$ c0_setup install
```
